# 체육시설 탐색 프로그램

 2022 한국공학대학교 스크립트언어를 이용한 텀프로젝트

# 소개
 - python을 이용한 각종 API 연동 및 XML을 이용한 정보 제공 프로그램 제
 - PC 어플리케이션
 - 2인 팀 프로젝트


# 개발환경
 - pycharm
 - C++
 - python
 - XML
